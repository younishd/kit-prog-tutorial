# kit-prog-tutorial

Slides for my Java programming tutorial at [KIT](https://kit.edu) using LaTeX beamer.

## Build

```bash
cd wise1617 # or wise1516
for i in *.tex; do pdflatex -output-directory=out $i; done
```
